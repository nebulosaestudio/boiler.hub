Que es una página web?

La página web es un archivo de texto, que interpretado por un navegador , muestra una representación visual multimedia que puede enlazar a otras páginas u otros contenidos.


Definición

Una página web es un soporte digital, que se caracterizan por que ademas de contener texto, y contenido multimedia como imágenes, videos , audio o documentos, pueden contener enlaces a otras paginas u objetos multimedia.

Las páginas web son archivos de tipo HTML, acronimo del Lenguaje de Marcado de Hipertexto ( Hypertext Markup Languaje ), este formato se caracteriza por contener texto plano, y permite introducirlo entre unas etiquetas definidas por el consorcio internacional w3c, que al ser interpretadas por el navegador, crean enlaces a otras páginas o documentos dentro de la web.

La representación visual que obtenemos en el navegador suele ser un conjunto de archivos (scripts)

Una página web se compone de :
  • HTML contenido.
  • CSS estilo.
  • JS interactividad.
Tipos de página web
  • newsletter
  • mail
  • landing page
  • mapa web
  • banner

Al conjunto de páginas sobre un tema determinado o bajo un dominio común se conoce como sitio web
